2. INBODY AI AUTO-EXTRACTION
==================================================

When a client uploads an InBody scan, AI/OCR must automatically read the scan and fill the InBody data fields.

The client should NOT have to enter the numbers manually.

Flow:
1. Client uploads InBody file.
2. System shows loading:
“Reading your InBody scan…”
3. AI/OCR extracts the numbers automatically.
4. Save the extracted data into the client’s InBody profile.
5. If confidence is low, mark it as:
Needs Review
6. Admin can review/edit manually.

Important:
- Do not guess missing values.
- If a value is not detected, show “Not available”.
- If AI extraction fails, registration/upload must still succeed.
- Show friendly message:
“We received your InBody scan. Youssef will review and update your analysis shortly.”

Extract these fields if available:
- Weight
- Body Fat %
- Skeletal Muscle Mass
- BMI
- Visceral Fat Level
- BMR
- Total Body Water
- InBody Score
- Body Fat Mass
- Segmental Lean Analysis
- Segmental Fat Analysis
- Test Date

6. PROTECTED CANCELLATION RULES
==================================================

Rename Emergency Cancel to:
Protected Cancellation

Rules by category:

Foundation Member:
- 0 per month

Progress Member:
- 1 per month

Elite Member:
- 2 per month

Protected Cancellation allows the client to cancel inside the 6-hour cutoff without losing the session.

If not available:
“This session is within the cancellation cutoff and will be counted as used.”

==================================================
7. SAME-DAY ADJUSTMENT RULES
==================================================

Rename reschedule feature to:
Same-Day Adjustment

Rules by category:

Foundation Member:
- 0 per month

Progress Member:
- 1 per month

Elite Member:
- 2 per month

Conditions:
- Can only move the session to another available time on the SAME DAY
- Must be at least 1 hour before the original session time
- Cannot move the session to another day

If client wants another day:
“To move this session to another day, you may use Protected Cancellation if available.”

If less than 1 hour:
“Same-Day Adjustment is no longer available. This session will be counted as used.”
