Deep mobile scroll-performance pass needed on the public homepage hero section.

Current issue:
When scrolling from the very top of the homepage (hero area), the motion feels heavy/slightly delayed on Android mobile Chrome. It is not silky/smooth yet. There is visible friction during the first scroll gesture from the hero area.

This is NOT a redesign request.
This is a pure rendering + motion-performance optimization pass.

Audit and optimize ALL possible causes of scroll jank, including:

1. Hero rendering performance

- Reduce expensive backdrop-blur usage
- Reduce layered gradients if too heavy
- Remove unnecessary opacity layers
- Avoid nested filters
- Avoid giant box-shadows
- Reduce paint complexity

2. Animation optimization

- Remove any scroll-linked animation causing layout thrashing
- Replace expensive animations with GPU-friendly transforms
- Use transform + opacity only
- Avoid animating height/top/left
- Ensure animations use will-change only where necessary
- Remove unnecessary transition-all usage

3. Mobile Chrome optimization

- Audit Android Chrome specifically
- Ensure smooth momentum scrolling
- Prevent repaint storms during first touch-scroll
- Reduce compositing overload

4. Image optimization

- Verify hero images are properly compressed
- Ensure responsive srcsets
- Lazy-load below-fold media only
- Prevent oversized background assets
- Avoid decoding large images during first scroll

5. CSS/layout optimization

- Eliminate layout shifts
- Prevent overflow repaint issues
- Reduce unnecessary sticky/fixed repaint cost
- Optimize viewport height handling on mobile
- Ensure no expensive CSS causing continuous recalculation

6. React/render optimization

- Prevent unnecessary rerenders on scroll
- Memoize expensive hero components if needed
- Avoid state updates tied to scroll movement

7. Final UX target
   The homepage should feel:

- buttery smooth
- premium
- lightweight
- iPhone-like fluidity
- zero friction on first scroll

Priority:
Smoothness > visual effects.

Do NOT break:

- existing design
- spacing
- branding
- animations globally
- RTL
- responsiveness

After optimization:

- run Lighthouse mobile performance audit
- verify on Android Chrome
- verify no FPS drops during first hero scroll
- verify no layout shift
- verify no stutter when pulling down/up quickly

Ship only after the homepage feels genuinely smooth on real mobile usage.
