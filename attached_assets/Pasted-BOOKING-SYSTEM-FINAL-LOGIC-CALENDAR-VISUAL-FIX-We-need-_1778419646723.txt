3. BOOKING LEAD TIME RULE — 3 HOURS

The booking lead-time rule must be exactly 3 hours.

Rule:

- Client cannot book any session that starts within the next 3 hours.
- Any session that starts more than 3 hours from now must be bookable if not already taken/blocked.
- Do NOT mark the entire day as Too Soon.
- Only the slots inside the 3-hour window should show Too Soon.

Example:
If current Dubai time is 5:00 PM:

- 5:00 PM = Too Soon
- 6:00 PM = Too Soon
- 7:00 PM = Too Soon
- 8:00 PM and later = available, unless taken/blocked

Important:

- Use Dubai timezone consistently.
- Avoid device timezone bugs.
- Confirm AM/PM parsing is correct.
- Confirm selected date + slot time creates correct Dubai datetime.
- Test today, tomorrow, and near-midnight cases.

4. STANDARD CANCELLATION RULE — 6 HOURS

Standard cancellation rule:

- Client cannot cancel normally if the session starts within the next 6 hours.
- If the session is more than 6 hours away, normal cancellation is allowed.
- When a normal cancellation succeeds, the slot must immediately become available again for any client.

Verify this is already working.
Patch only if broken.

5. EMERGENCY CANCEL + RESCHEDULE ELIGIBILITY

Eligibility must depend on client training frequency / package behavior.

Rules:
A) Clients with only 1 session per week:

- No emergency cancel allowance.
- No special reschedule allowance.
- Standard cancellation rule only.

B) Clients training 3 times per week:

- 1 emergency cancel allowance.
- 1 reschedule allowance.

C) Clients with more than 4 sessions / higher-frequency active package:

- eligible for emergency cancel/reschedule rules according to the same monthly allowance system.
- Keep allowance controlled and not unlimited.

Important:

- Emergency cancel and reschedule allowances must be limited.
- They must reset according to the intended monthly allowance logic.
- Do not allow unlimited emergency cancels or unlimited reschedules.

==================================================
6. EMERGENCY CANCEL BEHAVIOR

If a client uses emergency cancel:

- The session is cancelled.
- The client’s allowed emergency cancel count is consumed.
- The original time slot immediately becomes available to everyone.
- Any other client can book it.
- The same client can also rebook it only if still available.
- Priority is first-come, first-served.

The released slot is not reserved for the cancelling client.

==================================================
7. SAME-DAY RESCHEDULE RULE

Reschedule rule:

- Reschedule is allowed only on the same day.
- Reschedule depends on available slots on that same day.
- The client can move the session only to another available slot that day.
- The old slot is immediately released once reschedule succeeds.
- The new slot becomes taken.
- If there are no available valid same-day slots, reschedule is not possible.

Example:
If all later slots are already booked, and the last gym slot starts at 10:00 PM and ends at 11:00 PM:

- If no valid available slot exists, the client cannot reschedule.
- The session remains counted according to the rules.
- Do not create extra hidden slots beyond business hours.

==================================================
8. BUSINESS HOURS

Booking hours:

- first slot starts at 6:00 AM
- last slot starts at 10:00 PM
- last session ends at 11:00 PM
- no booking after 10:00 PM start time

All slot availability, reschedule, and cancellation logic must respect these hours.

==================================================
9. SLOT RELEASE PRIORITY

Whenever a booking is cancelled, emergency-cancelled, or successfully rescheduled:

- the old slot becomes available immediately.
- it can be booked by any client.
- no client owns the released slot until they book it again.
- first confirmed booking gets the slot.
