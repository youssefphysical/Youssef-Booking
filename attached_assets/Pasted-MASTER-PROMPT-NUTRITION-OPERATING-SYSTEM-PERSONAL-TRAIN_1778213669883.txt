PHASE 1 — SMART MACRO CALCULATOR
====================================================

Add a premium Macro Calculator tool inside admin.

Inputs:
- Protein grams
- Carbs grams
- Fats grams

Auto output:
- Protein calories = protein × 4
- Carbs calories = carbs × 4
- Fat calories = fats × 9
- Total calories
- Macro percentages
- Clean visual bars or rings

UI requirements:
- Premium cards
- Live auto-calculation
- Animated values
- Clear labels
- Mobile-first
- No page reload
- No console errors

Also add reverse helper if safe:
- enter total calories + target protein
- suggest remaining carbs/fats split

Do not make it medically complicated.
Keep it coach-friendly and practical.

====================================================

- meal notes

- quantity per food

  calories
  protein
  carbs
  fats

- Food values should scale correctly by quantity.

- Support meal alternatives later.

UI:
- meal card design
- food rows inside meal
- macro totals at top/bottom
- add/remove food smoothly
- mobile optimized
- avoid Excel-looking UI
