Required mobile-only fixes:

1. Remove expensive hero effects on mobile

- disable backdrop-filter on hero overlays
- disable large blur layers

- disable parallax/transform-on-scroll effects
- disable animated glow/sweep layers
- disable heavy shadows over the hero
- disable any infinite animations near hero

   On mobile:

- render the hero as a static image + simple gradient overlay only
- no moving hero layers
- no animated background elements
- no slider animation while scrolling
- no image filters

The visual should remain premium but performance comes first.

- lazy-load remaining slides

   No blur.
   No backdrop-filter.

5. Image optimization

- ensure mobile uses a true mobile-sized image, not desktop resolution
- set width/height or aspect-ratio to avoid layout shifts
- use object-fit: cover
- avoid decoding huge images during scroll
- preload only the first hero image

6. CSS containment

- contain: layout paint if it does not break sticky/header behavior
- content-visibility for below-the-fold sections if safe
