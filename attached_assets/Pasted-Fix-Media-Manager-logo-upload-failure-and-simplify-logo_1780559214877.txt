PART 1 — FIX UPLOAD PIPELINE

Required:

1. Do NOT write uploads directly to "/var/task/uploads/..." unless the directory is guaranteed to exist and writable.
2. Ensure upload directories are created before writing:
   - "/uploads"
   - "/uploads/brand"
   - "/uploads/media"
3. If running on Vercel/serverless production, do NOT rely on local filesystem persistence.
4. Use the project’s existing persistent storage solution for uploaded media.
5. If persistent object storage is not configured yet, store files safely in the existing working upload system or database-backed media storage.
6. Return a clear error if storage is unavailable.
7. Never show raw ENOENT errors to the admin.
8. Validate:
   - PNG
   - JPG/JPEG
   - WEBP
   - SVG only if safely sanitized
9. Max file size: 5 MB for logos.
10. Convert/optimize large images when possible.
11. Save media metadata:

- slot key
- asset URL
- filename
- mime type
- size
- dimensions
- updatedAt
- version/cache buster

13. Refresh preview without page reload.
